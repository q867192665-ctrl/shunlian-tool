# 瞬联调试工具 更新日志

## v1.0.5.1 (2026-05-24)

### 新增功能
- 新增在线更新功能，程序启动时自动检查新版本，发现更新时会弹窗提示下载

### 问题修复
- 修复无线配置中手动设置发射功率时提示错误的问题

### 界面优化

## v1.0.5 (2026-05-24)

### 新增功能
- 新增无线加密配置管理，可以添加、编辑、删除和开启/关闭加密配置
- 新增无线干扰扫描功能，快速查看各信道的干扰情况

### 优化改进
- 无线子菜单中不再显示多余的"刷新"按钮
- 加密配置的开启/关闭改为滑动开关，操作更直观
- 优化了错误提示，将看不懂的技术错误改成易懂的中文说明
- "开放认证"改名为"未启用认证"，表达更清楚
- 终端列表中信号弱的设备（≤-75dBm）会以橙色背景高亮显示
- 信号强度列同时显示发送和接收信号值，信息更完整
- 统一了各按钮的样式风格
- 无线配置页面支持折叠，高级配置、速率限制、组播等默认折叠收起

### 问题修复
- 修复了部分按钮颜色不协调的样式问题

---

## v1.0.4 (2026-05-17)

### 新增功能
- 新增设备重启功能，支持远程重启设备，操作时有二次确认提示
- 新增终端功能（New Terminal），可直接在工具中远程操作设备命令行

### 优化改进
- 修改非管理口的 IP 地址时，不再触发设备断开重连
- 编辑、添加、删除 IP 地址后，列表会自动刷新，无需手动操作
- 优化了对 7.x 固件设备的流量监控兼容性
- 终端页面切换时不会断开连接，输入历史和输出内容保留
- 优化了中文显示效果

### 问题修复
- 修复了 IP 地址页面有时搜索不到任何 IP 信息的问题
- 修复了编辑/添加 IP 时缺少备注填写功能的问题
- 修复了输入 IP 地址时无法正常输入点号的问题
- 修复了删除无线终端时弹窗风格不统一的问题
- 修复了重启确认弹窗中"取消"按钮不显示的问题
- 修复了发送重启命令后设备未实际重启的问题
- 修复了断线重连时弹窗显示无意义文字的问题
- 修复了设备重启后重新登录时默认停留在"重启"页面的问题
- 修复了终端页面显示空白、无法输入命令的问题
- 修复了切换菜单时终端连接会断开的问题

### 界面优化
- 优化了主菜单的滚动条样式，与整体风格更协调
