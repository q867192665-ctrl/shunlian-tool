# 瞬联调试工具 更新日志

## v1.0.4 (2026-05-17)

### 新增功能
- 新增系统-重启子菜单，支持设备远程重启功能，带二次确认弹窗
- 新增 New Terminal 终端功能

### 优化改进
- 优化 IP 地址修改逻辑，修改非管理接口 IP 不再触发设备重连
- 优化 IP 地址操作后数据刷新，编辑/添加/删除后立即更新前端列表
- 优化 7.x固件 设备流量监控，修复 `once` 参数格式兼容性问题
- 优化终端会话状态管理，支持跨页面切换保持连接和输出历史
- 优化中文字符显示
- 更新程序逻辑状态说明书，添加重启和终端功能相关定义

### 问题修复
- 修复 IP 地址界面搜索不到任何 IP 信息的问题
- 修复编辑/添加 IP 弹窗缺少注释功能的问题
- 修复 IP/子网输入拦截导致无法输入点号的问题
- 修复无线终端删除弹窗风格不统一的问题
- 修复重启二次确认弹窗"取消"按钮未显示的问题
- 修复设备重启命令发送后不生效的问题
- 修复断线重连弹窗显示无意义文本的问题
- 修复设备重启后重新登录默认处于"重启"子菜单的问题
- 修复 New Terminal 前端页面显示空白的问题
- 修复 New Terminal 无法输入任何字符的问题
- 修复 Telnet 会话在菜单切换时断开的问题

### UI 优化
- 优化主菜单竖向滚动条样式，与程序整体风格保持一致
