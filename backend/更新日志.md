# 瞬联调试工具 更新日志

## v1.0.9 (2026-06-27)

### 新增功能
- 带宽测速新增实时图表显示，替代原始文本输出
- 服务端反向测速模式自动检测：根据测速数据自动识别发送/接收方向

### 优化改进
- 移除双向测速开关，改为根据测速数据自动检测双向模式
- 反向测速仅对客户端模式生效，服务端模式自动适配
- 服务端模式图表自动显示为"接收"方向，客户端反向模式显示为"发送"方向
- 优化服务端停止测速响应速度

### 问题修复
- 修复测速时图表不显示导致页面空白的问题
- 修复服务端测速无法显示图表的问题
- 修复客户端图表下半部分被截断的问题
- 修复客户端反向测速结束后图表错误显示为"发送"的问题

### UI 优化
- 删除图表下方的数据统计卡片
- 图表图例文本"上传"改为"发送"，"下载"改为"接收"
- 服务端测速状态文字保持"测速中 (服务端模式)"不随测速完成变化
